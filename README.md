# IndeedBot 2026 - V1.0.0 RELEASE

The world's first multi-agent career intelligence infrastructure. De-risking your professional growth through adversarial protocol.

## Features
- **Intelligence Hub**: Market Extraction, Resume Alignment, Entity Scoring, Offline Persistence.
- **Next-Gen Tech**: Built with Next.js 16, Tailwind CSS 4, Framer Motion, and Lucide Icons.
- **Security First**: AES-256 E2E persistence and Zero-Track privacy protocols.

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure
- `src/app`: Next.js App Router pages and layouts.
- `src/components`: UI components built with a "rugged industrial" aesthetic.
- `src/app/globals.css`: Core design system and theme-aware styling.
